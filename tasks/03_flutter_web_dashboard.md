# Task 03: Web Dashboard Screens (Manager & Supervisor)

**Goal:** Build the command center views within the unified Flutter app.

## Sub-tasks
- [ ] **Master Data Management:**
  - Build views in `lib/screens/dashboard/` for managing `products` and `suppliers`.
- [ ] **PO & Shipment Tracking:**
  - Build dashboard to track active POs, Shipments, and Dock Doors.
- [ ] **Discrepancy Resolution & Operations:**
  - Build triage view for supervisors to review damaged/short flagged items and view uploaded photos.
  - UI to trigger "Recount/Audit" workflows for workers.
- [ ] **Import / Export / ERP:**
  - Implement `.xlsx`/`.csv` import for POs using web-compatible file pickers and parsers.
  - Implement GRN viewing and ERP sync status page.
- [ ] **Notifications:**
  - Implement real-time alerts for completed shipments and newly logged discrepancies.
